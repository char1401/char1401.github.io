Week 4 notes

<img>
The HTML element to display an image.
<a>
For creating a link on our webpage we use the anchor tag.
<thead>
The header of the table is used to add a heading row for the table columns. This is where we add labels for the data being displayed.
<form>
Form is a paired tag and any information we want to collect using this form will be added inside the opening <form> and closing <form> form tags.
<caption>
Every table can have a caption, this summarizes the data in the table.
